# 💰 Finly

A modern and minimalist personal finance dashboard to track income, expenses, goals and investments.

## 🚀 Tech Stack

**Frontend**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Recharts

**Backend**
- Laravel 11
- PHP 8.3
- MySQL
- Laravel Sanctum (auth)

## 🏛️ Architecture

Both frontend and backend follow **Clean Architecture** principles, with clear separation between entities, use cases, repositories and controllers.

## 📦 Project Structure

This repository is organized as a monorepo:

\`\`\`
finly/
├── frontend/   # Next.js application
└── backend/    # Laravel application
\`\`\`
